# Utils

## Propósito
Guardar funciones reutilizables y helpers de apoyo.

## Cuándo usarlo
- Formateadores.
- Validadores.
- Helpers de fechas, strings, objetos o arrays.

## Reglas
- Mantener funciones puras cuando sea posible.
- Evitar depender del contexto o de la UI.
- Usar nombres claros y específicos.
