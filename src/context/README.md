# Context

## Propósito
Compartir estado global o de alcance amplio entre varias partes de la app.

## Cuándo usarlo
- Autenticación.
- Preferencias de usuario.
- Estado global simple que no justifica una solución más pesada.

## Reglas
- Usarlo con moderación.
- Mantener los contextos pequeños y bien definidos.
- Evitar colocar toda la lógica del negocio en contexto.
