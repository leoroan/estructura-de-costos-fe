# Components

## Propósito
Contener componentes reutilizables y presentacionales.

## Cuándo usarlo
- Botones, formularios, tablas, tarjetas, modales y widgets compartidos.
- Componentes sin dependencias directas de negocio complejas.

## Reglas
- Mantenerlos pequeños y enfocados.
- Evitar lógica de negocio dentro de ellos.
- Preferir props claras y reutilizables.
