# Routes

## Propósito
Definir la estructura de navegación y el manejo de accesos.

## Cuándo usarlo
- Configurar vistas públicas y privadas.
- Aplicar protecciones de autenticación y redirecciones.

## Reglas
- Separar rutas públicas y privadas claramente.
- Mantener la configuración de navegación centralizada.
- Usar componentes de protección para encapsular reglas de acceso.
